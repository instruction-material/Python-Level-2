# Python-Level-2 Structure

This repository backs `Python Level 2` and selected Scratch course GitHub
examples from the live catalog. Historical folder names remain, but each active
project now uses the common wrapper shape.

Active public folders stay at the repository top level and contain
`README.md`, `starter/`, and `solution/`. Folders not linked by the live catalog
on 2026-05-14 were moved to `_archived-unlinked/`.

Some migrated projects have only one real legacy snapshot; the opposite role
folder contains a README placeholder until a distinct starter or solution is
authored.

New material should use the wrapper shape and should not add unlinked alternate
versions beside active folders.
