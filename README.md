# Python Level 2

Canonical source layout in this repo is folder-first: each project should live in
its own directory with any local assets beside it.

The old flat root-file layout is preserved under `_legacy-flat-layout/` so the
historical standalone files are still available without competing with the
current folder-based project structure.

`legacy-customized/` remains at the repo root because it preserves bespoke
historical variants outside the main canonical project layout.
